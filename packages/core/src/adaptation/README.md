## Relationship with InkOS

Monarch is built on [InkOS](https://github.com/Narcooo/inkos). For environment configuration, installation, and usage instructions, please refer to the [InkOS official repository](https://github.com/Narcooo/inkos).

Monarch shares the same environment configuration with InkOS and requires no separate setup.
